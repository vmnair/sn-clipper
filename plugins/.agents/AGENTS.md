# Project Rules & Guidelines

- **Plan Folder Custom Instructions & Assets**: 
  - Always check the `plan` folder within the project (`~/Projects/Supernote/plugins/sn-clipper/plan/`, which is a symbolic link pointing to `~/ai/Antigravity/Supernote/Plan/`) for user-provided instructions, icons, mockups, and assets.
  - When searching for files, always use commands that follow symbolic links (e.g. `find -L` or `ls -L`) because the `plan` directory is a symlink.
